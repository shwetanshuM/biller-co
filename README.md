millions of small businesses in india still track payments using registers, diaries, excel sheets, or just memory. someone always has to manually update records after every cash or upi payment — and it always gets missed, forgotten, or done wrong.

this is a chat-based business management tool that replaces all of that. owners can send casual messages like "rahul paid 1800 for july" and the ai understands it, updates the database, and replies with a confirmation — no app, no forms, just chat.

owners can also ask questions like "who hasn't paid this month", "how much did i collect today", "show me july's payments", or "who's been overdue for 2 months" and get instant answers — all through a simple chat interface.

currently building the backend with node, express, and mongodb. ai integration (natural language parsing via llm function calling) coming soon.

stack: node.js · express · mongodb · mongoose · jwt
